OBS setup

  1. In OBS, add a Browser Source.
  2. Paste URL: http://127.0.0.1:3847/overlay/obs (or run npm run obs:setup)
  3. Set size: 450 × 900 (adjust to taste).
  4. Check Shutdown source when not visible (optional).
  5. Enable transparent background in source properties if needed.

Kick blocked? Add KICK_SESSION_COOKIE to your .env file (copy from kick.com DevTools → Application → Cookies).
